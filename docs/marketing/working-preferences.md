# Working Preferences — Global Perspectives Marketing

## How to Work With This Project

### Output Format
- **Default output:** Markdown files, not chat text. Always produce a deliverable.
- **For copy (headlines, emails, ads):** Multiple options in a table — at least 3 variants per element
- **For longer content (blog posts, case studies):** Single best draft, structured with H2/H3 headers
- **For analysis (competitive, audience):** Tables where possible; bullets for lists of 4+

### Detail Level
- Concise is preferred over comprehensive
- Lead with the conclusion, follow with reasoning
- Do not pad copy with filler transitions ("In today's fast-paced world...", "Now more than ever...")
- Match length to platform — Twitter draft ≠ whitepaper paragraph

### Iteration Style
- Give the single best option first, then 2 alternatives if asked
- In one session, refine iteratively — don't restart
- When given feedback like "shorter" or "more direct," apply it immediately without re-explaining what you changed

### Constraints to Always Apply
- No exclamation points in analytical copy or long-form content
- No superlatives ("the best", "the only", "revolutionary") unless backed by a specific claim
- No emojis in written content (UI microcopy exception: arrows → and bullets • are fine)
- Always check against voice-profile.md tone before finalizing
- Don't invent statistics — flag where a stat is needed and leave a [DATA NEEDED] placeholder

### Platform Character Limits (Quick Reference)
| Platform | Headline | Description/Body |
|----------|----------|-----------------|
| Google Ads | 30 chars | 90 chars |
| Twitter/X | 280 chars total | — |
| LinkedIn | No hard limit | 1,300 chars before "see more" |
| Email subject | 50 chars ideal | 90 chars preview text |
| Meta (FB/IG) ad | 40 chars headline | 125 chars primary text |
| SEO title tag | 60 chars | — |
| SEO meta description | — | 160 chars |

### Review Checklist (Before Finalizing Any Asset)
- [ ] Does it match the voice in voice-profile.md?
- [ ] Is every factual claim accurate (no invented stats)?
- [ ] Does it communicate at least one of the 5 key messages from brand-context.md?
- [ ] Is the CTA specific and actionable?
- [ ] Has a human reviewed it before publishing?
